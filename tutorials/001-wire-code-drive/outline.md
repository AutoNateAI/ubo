# Recording Outline

1. Show the finished goal: a simulated robot car responding to real-style wiring and firmware.
2. Place the controller, motor driver, motors, and battery.
3. Explain why the controller does not power motors directly.
4. Wire power and common ground.
5. Wire motor outputs.
6. Wire direction pins.
7. Wire PWM speed pins.
8. Paste firmware and explain each helper function.
9. Run the simulation.
10. Debug one intentional issue or explain the debug order.
11. Connect the pin states to robot/game state.
12. Name the next challenge.
