# UBO Robotics Tutorials

These tutorials connect electronics, firmware, and simulator behavior. Each lesson should leave behind:

- a wiring goal
- a parts list
- a pin map
- code that could run on real hardware
- a simulator test plan
- debugging notes
- video/voice-over notes when useful
